# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-03-17

### Added
- Company logo (embedded as base64)
- Important legal disclaimers and notices
- FIFO queue policy guidelines for users
- User guidance for phone ordering after 11:00 AM

### Features Maintained
- JSONBin cloud persistence for cross-device sync
- Three-tier order status system (ממתין/בהכנה/מוכן)
- Warehouse staff management interface
- Admin panel with GPS controls and password management
- Order history and audit logging
- Recycle bin with restore functionality
- Drag-and-drop queue reordering

## [1.0.0] - 2026-03-17

### Added
- Initial stable release
- JSONBin cloud storage integration
- Async data loading before UI rendering
- Corrected API key authentication
- Full queue management system
- GPS verification with warehouse location
- Custom admin and staff password system
- Hebrew UI with RTL support
- Order status tracking with timestamps
